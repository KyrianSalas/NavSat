About NavSat

NavSat is an interactive real-time satellite tracking visualization platform. Using advanced orbital mechanics and Three.js 3D rendering, users can explore the orbits of hundreds of active satellites across Earth.

Built with precision, accuracy, and a focus on geopolitical awareness, NavSat displays satellites color-coded by their nation of origin, providing insight into global space activity.

Features

  • Real-time satellite position tracking with SGP4 propagation
  • Interactive 3D globe visualization using Three.js
  • Multiple satellite groups (Visual, Active, Stations, Weather, NOAA)
  • Detailed orbital parameters and statistics
  • Geopolitical color-coding by nation/agency
  • User location detection and visualization
  • Satellite-focused camera orbiting system
  • Orbital trail visualization with gradient effects

Technology Stack

  • Frontend: Vite, Three.js, satellite.js
  • Backend: FastAPI, Supabase PostgreSQL
  • Data Source: CelesTrak Orbital Database
  • 3D Graphics: WebGL with advanced shader effects

How It Works

NavSat fetches real-time TLE (Two-Line Element) data from CelesTrak, one of the most authoritative sources for satellite orbital information. Using the SGP4 propagation algorithm via satellite.js, we calculate precise satellite positions in real-time.

The visualization updates continuously as satellites orbit Earth, displaying accurate altitudes, velocities, and orbital parameters. Click on any satellite to focus the camera and view detailed statistics.

Project Origins

NavSat was developed for BrisHack 2026 as an exploration into the intersection of space technology, data visualization, and geopolitical awareness. The project demonstrates how modern web technologies can make complex space data accessible and engaging for everyone.
